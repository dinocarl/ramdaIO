# ramdaIO

Simple means of creating an environment similar to the [ramda REPL] using node
and a console.

## Setup

```bash
npm install
```

## Use it

1. Add a file
2. Point `src/buffer.js` at that file
3. Run `npm run watch`
4. Make changes and see the console update

  [ramda REPL]: https://ramdajs.com/repl/
